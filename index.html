🎉 Personalized Calendar

Pick a theme, or manage your dates, quotes & photos in one place.

Welcome 👋

This is a self-hosted, printable birthday & anniversary planner built with plain HTML, CSS and vanilla JavaScript — no frameworks, no build step.

  • Choose a theme below to open a printable calendar.
  • Manage Data lets you add / edit dates, quotes and month photos.
  • Your changes are saved in the browser and applied to every theme automatically.
